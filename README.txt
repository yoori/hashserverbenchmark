Тест для замера производительности HashServer'а
не проверяет корректность содержимого ответов (только кол-во по разделителю)

Компиляция (стандартная при использовании cmake):
mkdir build
cd build
cmake ..
make

Запуск:
HashServerBenchmark -H <ip адрес сервера> -p <порт на котором запущен сервер> -c <кол-во потоков> -r <кол-во запросов>

Пример:
HashServerBenchmark -H 127.0.0.1 -p 10001 -c 20 -r 10000000
